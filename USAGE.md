# 使用说明

## 快速开始

### 1. 启动服务器
```bash
./start.sh
```

### 2. 访问应用
- 打开浏览器访问: http://localhost:8080
- 或者直接访问:
  - 图片切分工具: http://localhost:8080/pic/
  - 安全文本传输: http://localhost:8080/copy/

## 安全文本传输工具详细使用

### 步骤1: 设置密钥
1. 在"设置密钥"输入框中输入密钥
2. 密钥只能包含大小写字母和数字
3. 点击"设置密钥"按钮
4. 密钥设置后不可编辑，只能全部删除后重新输入

### 步骤2: 建立连接
1. 密钥设置成功后，系统会自动连接WebSocket服务器
2. 连接状态会显示在"连接状态"区域
3. 绿色圆点表示已连接，红色表示未连接
4. 支持自动重连功能

### 步骤3: 发送消息
1. 在"发送消息"文本框中输入要传输的内容
2. 点击"发送消息"按钮或按Ctrl+Enter
3. 消息会使用密钥进行AES加密后发送
4. 发送成功后会显示确认消息

### 步骤4: 接收消息
1. 相同密钥的所有客户端都会收到加密消息
2. 消息会自动解密并显示在"接收到的消息"区域
3. 每条消息都显示接收时间
4. 支持多客户端同时在线

## 多客户端测试

### 测试步骤
1. 打开多个浏览器标签页或窗口
2. 都访问 http://localhost:8080/copy/
3. 设置相同的密钥
4. 在任意一个客户端发送消息
5. 其他客户端会收到相同的消息

### 注意事项
- 只有使用相同密钥的客户端才能互相通信
- 不同密钥的客户端之间无法通信
- 密钥区分大小写
- 建议使用复杂的密钥以确保安全性

## 故障排除

### 连接问题
- 检查服务器是否正在运行
- 确认端口8080未被占用
- 检查防火墙设置

### 消息传输问题
- 确认所有客户端使用相同密钥
- 检查WebSocket连接状态
- 刷新页面重新连接

### 加密问题
- 确认密钥只包含字母和数字
- 检查浏览器控制台是否有错误信息
- 尝试重新设置密钥

## 技术细节

### 加密算法
- 使用AES加密算法
- 密钥长度: 用户自定义
- 加密模式: CBC

### 通信协议
- WebSocket协议
- JSON消息格式
- 自动重连机制

### 安全特性
- 密钥不存储在服务器
- 使用MD5哈希标识客户端
- 端到端加密传输
